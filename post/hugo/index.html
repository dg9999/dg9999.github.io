  • 下载Hugo

https://github.com/gohugoio/hugo/releases

下载完需要配置环境变量

  • 生成站点

hugo new site myblog(一个文件夹的名字)

  • 下载主题

/myblog git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

  • 本地启动

hugo server -t m10c –buildDrafts

  • 部署到远端

hugo –theme=m10c –baseUrl=“https://dg9999.github.io" –buildDrafts

会生成一个public文件夹

  • 推到github仓库

hugo –theme=m10c –baseUrl=“https://dg9999.github.io" –buildDrafts

git init

git add .

git commit -m “我的hugo博客第一次提交”

git remote add origin https://dg9999.github.io.git

git push 不行了话(git push -u origin master)
